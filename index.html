About the Project

  • This sample app mimics the structure of our larger app
  • Prepare for an easy migration from ng1 to ng2
  • Applies ng2 concepts/best practices to an ng1 app today

Characteristics of the Project

  • Written in Angular 1.5
  • Writen in TypeScript
  • Utilizes ES6 Classes
  • Component Architecture
  • One-way data binding
  • Lots of Unit Tests
  • No .controller and No $scope
